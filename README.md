# Flexoki Dark - Theme for Omarchy

## Installation:

### Method 1
1. Copy `https://github.com/euandeas/omarchy-flexoki-dark-theme`
2. Click `SUPER + ALT + SPACE`
3. Click `Install`
4. Click `Theme`
5. Paste the link 

### Method 2:
```bash
omarchy-theme-install https://github.com/euandeas/omarchy-flexoki-dark-theme.git
```
